> 凑十法与破十法、数感建立、图形认知——用生活场景和动手操作代替枯燥练习，让孩子对数学产生好奇心。

## 启蒙节奏建议

数学启蒙的核心不是计算速度，而是数感——孩子能理解"数量"和"关系"。建议用积木、扑克牌、水果等实物操作，每天 10-15 分钟游戏化练习，避免纸面刷题。
