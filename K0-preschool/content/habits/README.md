# 习惯养成

> 习惯比知识更重要。本模块覆盖专注力、握笔坐姿、物品管理和社交适应四个方向，帮孩子建立"准备好上学"的状态。

## 为什么习惯最重要

坐得住、管得了自己的东西、能听懂并执行老师的指令——这些比多认几个字更决定入学后的适应速度。建议从入学前 6-12 个月开始培养，每天 15-20 分钟即可。
