# 学途地图 (EduAtlas)

**面向家长的结构化教育知识库** — 用一张地图，看清从幼小衔接到高中毕业的完整学习路径。

不是教辅资料，不是题库，而是帮你**看清全局、抓住重点、避开弯路**的教育导航系统。

## 这是什么

学途地图把零散的教育知识整理成结构化的知识体系，用流程图和清单帮你回答三个核心问题：

1. **该准备什么** — 每个阶段的核心知识点和能力要求
2. **先抓什么** — 按重要程度分级，时间有限只看最重要的
3. **容易错在哪** — 43 条易错点汇总，辅导时直接翻

所有内容基于国家课程标准和公开知识整理，不搬运教材原图/原题。

## 如何使用

1. **第一次来**：打开 [K0 幼小衔接总览](K0-preschool/README.md)，花 10 分钟了解全貌
2. **找重点**：翻 [重难点速查](indexes/key-points.md)，快速抓住核心
3. **辅导孩子时**：翻 [易错点清单](indexes/common-mistakes.md)，知道哪里容易出错
4. **想看全局**：打开 [全览地图](OVERVIEW.md)，了解从幼小衔接到高中的完整体系

所有文件都是 Markdown 格式，在 GitHub 上直接阅读即可，手机和电脑都能看。
