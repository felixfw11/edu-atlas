# edu-atlas

EduAtlas 项目的结构化内容产品仓库。

## 项目定位

本仓库负责：

1. **内容存储**：经过校验的 `.md` 与 `.mmd` 结构化知识文件
2. **分级管理**：按阶段组织内容，覆盖幼小衔接 → 小学 → 初中 → 高中全程
3. **产品交付**：结构化知识库的源文件管理

## 阶段编码说明

| 前缀 | 含义 | 范围 |
|------|------|------|
| K | 小学（含幼小衔接） | K0 幼小衔接，K1-K6 一至六年级 |
| J | 初中（Junior） | J1-J3 七至九年级 |
| S | 高中（Senior） | S1-S3 高一至高三 |

每个阶段按学期细分：`-s1`（上学期）、`-s2`（下学期），K0 为特殊阶段无学期划分。

## 产品矩阵

| 阶段 | 内容 | 状态 |
|------|------|------|
| K0 | 幼小衔接 | 已完成 |
| K1-K6 | 小学一至六年级（12 个学期） | 待开发 |
| J1-J3 | 初中七至九年级（6 个学期） | 远期规划 |
| S1-S3 | 高中高一至高三（6 个学期） | 远期规划 |

## 内容规范

所有内容文件须遵循统一的：

- Markdown 样式标准
- YAML 元数据规范
- Mermaid 图表规范

## 版本管理策略

| 要素 | 说明 |
|------|------|
| `main` | 成品分支，禁止直接 push，通过 PR 合并 |
| `v0.x.y` | 开发分支，日常工作在此进行 |
| 重大调整 | 创建新分支（如 `v0.2.0`），旧分支保留不删除 |
| 版本号格式 | `v主版本.里程碑.补丁`（语义化版本） |
